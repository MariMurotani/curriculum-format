# 設計
どういったアプリケーションを作成していくのか、アプリケーションの全体像を掴めるように設計のパートを記載していただければと思います。

例えば、「[Instagram風簡易SNSアプリを作ってみよう【Rails】](https://github.com/Techpit-Contents/techpitgram3)」という教材では、全画面の紹介とテーブル設計を記載しています。
